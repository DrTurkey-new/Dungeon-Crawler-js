# 🗡️ Dungeon Adventure: Console Edition

A text-based RPG adventure game built with **JavaScript**! This was my first major project completed during **Week 3** of learning JS.

## 🕹️ How to Play
1. Open `Game.html` in any web browser.
2. **Press F12** on your keyboard (or Right-Click > Inspect).
3. Click on the **'Console'** tab.
4. Follow the prompts to enter your name and starting health!

## ⚔️ Game Features
- **Randomized Gear:** Start with a Wooden Spoon, Stone Sword, Golden Axe, or the legendary Bamboo Stick.
- **Dynamic Encounters:** Face off against a Rubber Duck, Wild Boar, Dragon, or the rare Baby Panda.
- **Logic-Based Combat:** Damage is calculated based on which weapon you are holding against which boss.
- **RNG Health:** Your starting HP is randomly generated between 70 and 100.

## 🛠️ Built With
- **HTML5**: The game's structure and "Play Again" functionality.
- **JavaScript**: Core logic using `Math.random()`, `switch` statements, and nested `if/else` conditions.
